Set Defination
The collection of well defined object is called Set. It is denoted by capital letters and elements / members are denoted by small numbers.
1) A = {a,b,c,d}
2) B = {x:x is a prime numbers less than 7}
3) C = The set of even numbers less than 10

Representation of Set
1) Listing Method: A = {1, 2, 3, 4, 5}
2) Set - Builder Form: A = {x:x is natural numbers less than 6}
3) Description Method: B = The set of natural numbers less than 6

Types of Sets
1) Null Set / Empty Set
2) Singleton Set
3) Overlapping and Disjoint Set
4) Finite and Infinite Set
5) Universal Set

Relation on Set
1) Subset
2) Equal Sets
3) Equivalent Sets
4) Intersection and Union of Set
5) Overlapping and Disjoint Sets

Power Set
If S is a non empty set than collection of all subset of S is called power set of S. It is denoted by p(s) or 2n

Proper Subset ← Subset → Improper Subset
A = {a, b}
Proper Subset = {a}, {b}
Improper Subset = {a, b}

Operation on Sets
1) Union of Sets: AUB = {X : X∈A and X∈B}
2) Intersection of sets: AUB = {X : X∈A and X∈B}

Question No.1
If ∩ (U) = 200, ∩ (A) = 85, ∩ (B) = 60, ∩ (AUB) = 100 then,

Find ∩ (A∩B), ∩ (A-B) And ∩ (AUB)

Solution here,

We know that

∩ (AUB) = ∩(A) + ∩ (B) - ∩ (A∩B)

or, 100 = 85 + 60 - ∩ (A∩B)

or, 100 = 145 - ∩ (A∩B)

or, 100 - 145 = - ∩ (A∩B)

or, - 45 = - ∩ (A∩B)

45 = ∩ (A∩B)

Therefore, ∩ (A∩B) = 45


Again,

∩ (A - B) = ∩o (A)

= ∩ (A) - ∩ (A∩B)

= 85 - 45

= 40


At Last,

∩ (AUB) = ∩ (U) - ∩(AUB)

= 200 - 100

= 100
