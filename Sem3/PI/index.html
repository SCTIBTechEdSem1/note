Principles of Internet Technologies and Web Applications

Unit 1: Web Fundamentals (5 Hours)
1.1 The Internet and the Web
i. Internet vs. World Wide Web.
ii. Brief history and evolution of the web.
iii. Web pages, websites, and web applications.
iv. Web 1.0, Web 2.0 and Web 3.0.

1.2 URLs, Domains, and DNS
i. Structure of a URL.
ii. Domain names and DNS lookup.
iii. Role of DNS in reaching websites.

1.3 Client–Server Architecture
i. Basic client–server model.
ii. 1-tier, 2-tier, and 3-tier architectures.
iii. Static vs. dynamic content.
iv. High-level view of modern web app architecture.

1.4 Core Internet and Web Protocols
i. TCP/IP and UDP.
ii. HTTP request–response cycle.
iii. Introduction to HTTPS and TLS/SSL.

1.5 APIs and JSON in Modern Web Applications
i. What an API is and why modern applications rely on them?
ii. REST-style thinking at a high level.
iii. JSON vs. HTML vs. XML: what each is used for.
iv. Simple example of a JSON response from a web service.

Unit 2: Hyper Text Markup Language (7 Hours)
2.1 Introduction to HTML and Document Structure
i. Introduction to HTML:
a. Overview of HTML's role in web development and the client-side structure of web pages.
b. The function of HTML in creating structured web content.
ii. Structure of an HTML document including meta tags, elements and attributes, writing Comments.

2.2 Basic HTML Elements and Text Formatting
i. Headings, Text Formatting: Paragraphs and Preformatted Text elements, Presentational Elements, Phrase Elements, Editing Elements, Grouping Elements.
• Build tables, lists, and embed external content such as maps or videos using iframes.
• Construct forms using common form controls and apply basic HTML5 validation attributes.
• Work with multimedia elements and apply semantic HTML5 sectioning tags to improve structure and accessibility.

2.3 Hyperlinks, Images, and URL Types
i. Hyperlinks:
a. The <a> element for linking.
b. Difference between absolute and relative URL with examples.
ii. Images:
a. Embedding images with the <img> tag.
b. Difference between ismap and usemap attributes with examples.

2.4 Tables, Lists, and Embedding External Content
i. Tables and Nested tables with attributes like colspan and rowspan.
ii. Lists:
a. Unordered lists, ordered lists, and definition lists
iii. Iframes for Embedding External Content:
a. Embedding external content using <iframe>.
b. Embedding Google Maps, YouTube videos, and other web pages.

2.5 Forms and Form Elements
i. Forms Overview:
a. Using <form> to create user input forms.
b. Form attributes: action, method and enctype.
ii. Form Elements: a. Input Elements: <input> types, <textarea>, <select>, <option>;, <button>, <label>. b. Fieldset and Legend: Grouping form elements with <fieldset> and adding a caption with <legend>.
iii. Validation:
a. HTML5 input validation: Required fields, pattern matching with pattern, and range control.


2.6 Multimedia and HTML5 Sectioning Elements
i. Working with Multimedia:
a. Embedding audio and video using <audio> and <video> tags.
b. Audio/video attributes: controls, autoplay, loop, and handling supported formats.
ii. HTML5 Sectioning Elements:
a. Using semantic HTML5 elements to structure
content: <article>, <section>, <nav>, <header>, <footer>, <aside>.
